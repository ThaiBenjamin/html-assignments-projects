## 📚 What I Was Learning

Foundational front-end web development — writing semantic HTML, applying CSS styling rules, using the Bootstrap framework for responsive layouts, and writing basic JavaScript for interactivity.

---

## 🔑 Key Things Practiced

- **HTML Structure** — Semantic elements, forms, tables, links, images
- **CSS Styling** — Selectors, the box model, colors, typography, positioning
- **Bootstrap** — Grid system, utility classes, pre-built components (navbars, cards, modals)
- **JavaScript Basics** — DOM manipulation, event listeners, conditionals, functions
- **Assignment-Driven Learning** — Each folder is a discrete task with a specific learning goal

## 💡 What It Taught Me

Starting from scratch with HTML and CSS made me appreciate how the web actually works beneath all the frameworks. I learned that getting layout right requires understanding the cascade and the box model deeply, not just copying properties until something looks right. Bootstrap showed me how to build responsive pages quickly, but also made me want to understand what it was doing under the hood. By the time I got to JavaScript, I realized how much of web interactivity comes down to listening for user events and updating the DOM in response.
